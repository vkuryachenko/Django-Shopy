Django-Shopy

Django-based E-comerce platform
with friendly admin interface.



Django-Shopy has been developed as a free open-source alternative of well-know SaaS e-comerce platfrom shopify.com.
It allows you to build your shop based on popular python framework Django and use multiple ready e-comerce themes. In compare with other Django shops Shopifies has nice UI for staff user which is based on Andgular 2.



Now

Project is in deep pre-alpha version for the moment, so already supported:

  • Admin interface for staff users, products and customers.



What is planned:

  • full support of liquid template engine
  • Import and export with shopify.com
  • Support of localization and internalization
  • and many othersemail
